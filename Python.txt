CONDA
Creating new project virtual env: (Recommendated for self contained - and not to run into issus with packages)
conda create --MyDjangoProjectEnv django
activate MyDjangoProjectEnv

Working with DJagno
conda install django / pip install django
django-admin startproject first_project

Run server
cd first_project\
python manage.py runserver

Creating the app
C:\..\first_project> python manage.py startup first_app
Open settings.py and add first_app in INSTALLED_APPS
